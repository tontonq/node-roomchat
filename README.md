## To install

    git clone https://github.com/tontonq/node-roomchat.git
    cd node-roomchat
    npm install
    node app.js

## Usage & Info

This project used Node.js , Socket.io , Express 3.0 , Jade
just make the right configuration  between Settings tags

then run
    node app.js
